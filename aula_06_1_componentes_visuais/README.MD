## Exercicios da Aula

- Criar uma tela de cadastro de aluno com os seguintes campos:
  - Nome
  - E-mail
  - Telefone
  - Data de Nascimento
  - Cidade
  - Estado
  - CEP
  - Endereço
  - Número
  - Complemento
  - Bairro
- Incluir um botão para simular o cadastro do aluno
- Ao clicar no botão deve emitir um alerta com a mensagem "Aluno cadastrado com sucesso"
